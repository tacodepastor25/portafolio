API History

Funciones de history

La API History de HTML te permite interactuar con la historia del navegador del usuario, lo que significa que puedes controlar el botón "Atrás" y "Adelante" del navegador.

Este ejemplo muestra cómo utilizar la API History para agregar un nuevo estado a la historia del navegador cuando se hace clic en un botón. Luego, utiliza el evento onpopstate para detectar cuándo el usuario ha navegado hacia atrás o hacia adelante en la historia. En este caso, el evento muestra una alerta dependiendo del estado al que el usuario haya navegado.

  • Funciones comunes
  • history.length: Esta propiedad devuelve la cantidad de estados en la historia del navegador.
  • history.forward(): Esta función hace que el navegador avance al siguiente estado en la historia.
  • history.back(): Esta función hace que el navegador vuelva al estado anterior en la historia.
  • history.go(delta): Esta función te permite navegar a un estado específico en la historia. El parámetro delta especifica la cantidad de estados que se deben mover hacia atrás (valores negativos) o hacia adelante (valores positivos) en la historia.
